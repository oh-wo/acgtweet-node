# ACGTweet-node

## Requirements
- [node]()
- [npm]() (installed with node)

## Installation
 - `npm install`
 
## development
- To start the app locally `node index.js` or to watch and hot reload: `nodemon index.js`

## useful psql commands
- `\list` list all databases
- `\dt`   list all tables in the current database
